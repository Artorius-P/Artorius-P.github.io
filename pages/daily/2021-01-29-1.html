      # Linux下查看硬件温度

      来自博客 (opens new window)

      # 安装

      主要使用lm-sensors hddtemp 这两个软件 在Ubuntu下

      sudo apt-get install lm-sensors hddtemp
sudo sensors-detect

      # 使用

      命令行输入sensors即可使用。
